1、解压文件时报错           tar: Error is not recoverable: exiting now

​      解决方法:去掉解压文件中的j即可      

```shell
tar -jxvf libspicy.tar.bz
tar -xvf libspicy.tar.bz
tar cvf  test.tar 1.txt 2.txt 3.txt
```

 2 、归档管理

   + -c  创建打包文件

   + -v   显示归档，解压缩时的进度

   + -f  指定名称，必须是.tar 文件，所以f必须要放在最后

   + -t  列出档案中的文件

   + -x  解开档案文件
