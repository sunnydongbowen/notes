### 1、cat  查看文件所有内容

​       合并文件内容

```shell
cat 1.py 2.py  > 3.py  # 合并文件内容到3，谁在前面谁就先写进去。
ls > 1.txt   # 把ls的输出写到1.txt里  【如果1.txt 不存在，就自动创建】，清空源文件再追加
ls >> 1.txt  # 直接追加，不覆盖。在后面追加。
```

### 2、 |  管道符

​     前一个命令的输出通过管道符号【|】作为后一个命令的输入

```shell
ps -ef|grep java
ll | more
```
